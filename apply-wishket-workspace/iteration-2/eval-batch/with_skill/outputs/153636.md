- 결과: 비전문가도 자연어로 시뮬레이션 파이프라인 조작 가능 — 사용 진입 장벽 대폭 감소

2) B2B SaaS 관리 플랫폼 — 프로토타입→상용 전환 + 관리자 기능
- 프로토타입 수준 코드베이스를 상용 아키텍처로 재설계한 경험, 관리자 대시보드 및 회원/권한 관리 시스템 구축
- 실제로 겪은 이슈: 동일 기능이 페이지마다 제각각 구현되어 수정 시 10곳 이상 변경 필요 → 53개 공통 컴포넌트 디자인 시스템을 구축하여 표준화
- 결과: 공통 컴포넌트 조합만으로 신규 페이지 구성 — 개발 시간 50% 단축, 이 구조 위에 6개 관리 모듈 안정적 추가 완료
