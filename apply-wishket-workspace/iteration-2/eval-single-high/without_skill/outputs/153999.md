## 관련 경험

### 1. 출근노트 — B2B SaaS 어드민 시스템 (4년 설계 및 운영)

본 프로젝트와 가장 직접적으로 관련된 경험입니다. React/TypeScript 기반 B2B2C 근태/급여 관리 SaaS의 프론트엔드를 처음부터 설계하고 4년간 운영했습니다.

**어드민 UI 구현 경험:**
- Chakra UI 기반 **53개 공통 컴포넌트** 직접 설계 및 추출 (버튼, 입력 폼, 테이블, 모달, 탭, 검색 필터 등)
- 어드민 → 노무법인 → 사업주 → 중간관리자 → 근로자 **5단계 권한 체계** 어드민 시스템 구축
- 노무법인 어드민 시스템 **6개 모듈 신규 개발**: 대시보드, 부관리자 관리, 기본정보 관리, 계정 찾기, 근로자 정보 관리, 사업장 검색
- 수천 명 규모 급여 정산 테이블 — react-window 가상화 + React.memo로 **UI 프리징 완전 해소**
- 컴포넌트 조합만으로 신규 페이지 구성 가능한 체계 구축, **개발 시간 50% 단축**

**기술 스택:** React, TypeScript, Jotai, Zustand, React Query, Zod, Chakra UI

### 2. 에브리드론 — Next.js App Router 기반 B2B SaaS (현재 재직중)

현재 Next.js(App Router) + TypeScript 기반 드론/CFD 시뮬레이션 플랫폼을 개발하고 있습니다.

- Next.js App Router 기반 프로젝트 실무 경험
- Storybook 기반 디자인 시스템 + 컴포넌트 인터랙션 테스트 자동화
- PostHog 분석, Prisma ORM, 모노레포 아키텍처 설계

**기술 스택:** Next.js, TypeScript, Nest.js, Prisma, Storybook, PostHog

### 3. 포스코 MES — 대량 데이터 테이블 처리

Electron + React 기반 실시간 대시보드에서 WebSocket 대량 데이터를 테이블로 표시하는 작업을 수행했습니다.

- 데이터 전처리 필터링 + 배치 렌더링으로 DOM 업데이트 최소화
- 규칙 엔진 기반 설정 화면 설계 — 코드 수정 없이 조건 변경 가능
