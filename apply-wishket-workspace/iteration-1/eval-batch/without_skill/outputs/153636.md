## 적합성

### 1. 채팅 서비스 개발 및 안정화 경험

**에브리드론 — AI SDK v5 기반 챗봇 프론트엔드 (현재 재직중)**
- AI SDK v5 기반 챗봇 구축 — 스트리밍 응답 렌더링, 클라이언트 사이드 도구 실행으로 시뮬레이션 직접 제어
- 대화 히스토리 관리, 실시간 스트리밍, 도구 실행 상태 표시 등 채팅 UI/UX 전반 구현
- AI 캐릭터 채팅 플랫폼과 유사한 실시간 메시지 처리 + 상태 관리 패턴을 직접 설계·운영 중

**포스코 MES (GSS) — 실시간 WebSocket 데이터 처리**
- WebSocket 기반 대량 데이터 실시간 처리 — 데이터 유입 속도가 클라이언트 가공 속도를 초과하는 문제 해결
- 전처리 필터링 + 배치 렌더링으로 DOM 업데이트 최소화, 실시간 처리 성능 확보

### 2. 기존 코드 구조 파악 및 개선 능력

**에브리드론 — 프로토타입 → 상용 아키텍처 재설계**
- 프로토타입 수준 코드베이스를 상용 아키텍처로 전환: WASM-JS 계층 분리, 디자인 시스템 표준화, 인증·인프라 체계 구축
- 통합 컬러 팔레트 시스템 + shadcn 컴포넌트 재구성으로 디자인 시스템 표준화
- NextAuth 인증, Devcontainer, nginx 보안 헤더, GitHub Actions 등 인프라 정비

**오픈소스 23개 프로젝트 기여**
- laravel-mix (5.2k stars)에 12개 PR merged — webpack 내부까지 분석하여 React Fast Refresh 구현
- CLIProxyAPI (18.8k stars)에 5개 PR merged — 남의 코드를 빠르게 읽고 핵심 기능 기여
- 이처럼 외부 코드베이스를 분석하고 개선하는 것이 일상적인 작업 방식

### 3. PG 결제 연동 경험

- 토스페이먼츠 연동 경험 보유 — 결제 흐름(주문 생성 → PG 승인 → 웹훅 검증 → 상태 갱신) 전체 이해
- 출근노트 B2B SaaS에서 결제 모듈 연동·운영 경험

### 4. 관련 기술 스택

| 요구 기술 | 보유 경험 |
|-----------|-----------|
| Remix | React 기반 프레임워크 — Next.js 4년+ 경험으로 빠르게 적응 가능, loader/action 패턴 이해 |
| Supabase | PostgreSQL 기반 BaaS — PostgreSQL 직접 운영 경험, Supabase Auth/Realtime/Storage API 활용 가능 |
| PostgreSQL | Nest.js + Prisma + PostgreSQL 현재 운영 중 |
| Cloudflare | Workers, Pages, R2 등 Cloudflare 인프라 활용 경험 |
