## 관련 경험 및 역량

### 1. 멀티플랫폼 개발 및 출시 경험

**출근노트 (B2B SaaS, 2020-2025)** — React Native + WebView 하이브리드 구조로 웹/Android/iOS 3개 플랫폼을 단일 코드베이스로 서비스하고 스토어 배포까지 완료했습니다. Platform.OS 기반 추상화 레이어로 플랫폼별 분기를 격리하여 비즈니스 로직을 공유하면서 네이티브 연동부만 분리하는 설계를 적용했습니다.

> 본 프로젝트의 Android/iOS 앱 개발 및 백그라운드 음성 채팅 유지 요구사항에 직접 적용 가능합니다.

### 2. 실시간 데이터 처리 경험

**포스코 MES (GSS, 2019-2023)** — WebSocket 기반 실시간 철판 결함 데이터 처리 시스템을 개발했습니다. WebSocket 데이터 유입 속도가 클라이언트 가공 속도를 초과하는 문제를 전처리 필터링 + 배치 렌더링으로 해결하여, 데이터 유입 속도에 밀리지 않는 실시간 처리 성능을 확보했습니다.

> WebRTC 시그널링 및 실시간 음성 스트림 처리에 필요한 실시간 통신 아키텍처 설계 역량을 보유하고 있습니다.

### 3. 대규모 사용자 대상 성능 최적화

**출근노트** — 수천 명 규모 사업장의 급여 정산 화면에서 UI 프리징 문제를 react-window 가상화 + React.memo + Redis 캐싱 + N+1 쿼리 해소로 프론트/백엔드 양단 병목을 제거했습니다.

> 다수 사용자가 동시 접속하는 음성 채팅 서비스에서 필수적인 성능 최적화 경험입니다.

### 4. 풀스택 아키텍처 설계

- **프론트엔드 설계 시스템 구축**: Chakra UI 기반 53개 공통 컴포넌트 디자인 시스템으로 개발 시간 50% 단축
- **상태관리 아키텍처**: React Query/Zustand/Jotai 3계층 상태관리 설계 — 서버/전역/로컬 상태 분리
- **권한 체계**: B2B2C 5단계 권한 체계 설계 — PermissionGate 패턴으로 중앙 집중화
- **백엔드/인프라**: Laravel/Nest.js 백엔드, Docker, GitHub Actions CI/CD, AWS 인프라 운영

### 5. 기획부터 개발까지 전 과정 수행

**에브리드론 (2025-)** — 프로토타입 수준 코드베이스를 상용 아키텍처로 재설계하여 v1.0 릴리즈를 완료했습니다. WASM-JS 계층 분리, 디자인 시스템 표준화, 인증/인프라 체계 구축 등 제품 전 영역의 설계와 개발을 수행하고 있습니다.

## 기술 스택 제안

| 영역 | 기술 | 선택 이유 |
|------|------|-----------|
| **프론트엔드 (Web)** | Next.js + TypeScript | SSR/SSG 지원, React 생태계 활용, SEO |
| **모바일 앱** | React Native | 웹 코드 재사용 극대화, iOS/Android 동시 대응 |

| **백엔드** | Nest.js + TypeScript | 모듈 기반 아키텍처, WebSocket 네이티브 지원 |
| **DB** | PostgreSQL + Redis | 관계형 데이터 + 세션/캐시/Pub-Sub |
| **인프라** | AWS (ECS/RDS/ElastiCache) | 확장성, 관리형 서비스 |
| **결제** | 토스페이먼츠 / 아임포트 | 국내 PG 연동 편의성 |
| **관리자** | Next.js + shadcn/ui | 통계 대시보드, 회원/결제 관리 |
