## 클라이언트 질문 답변

> 동시 접속 최대 인원 / 아지트의 상시 존재 여부

동시 음성 참여 인원에 따라 아키텍처가 달라집니다:
- 4~6명: P2P(mesh) 방식으로도 가능, 서버 비용 최소
- 10~20명: SFU(Selective Forwarding Unit) 서버 필수 — LiveKit, mediasoup 등
- 50명+: 상용 SaaS(Agora, Twilio) 권장

"아지트 상시 존재"는 서버 비용과 직결됩니다. 아무도 접속하지 않은 아지트는 DB 레코드만 유지하고 미디어 서버 리소스는 해제하는 구조가 일반적입니다. 미팅에서 상세 논의하면 좋겠습니다.

### 금액/기간 산정

**Step 1: 스코프**
- 기능 모듈 12개, 업무 범위: 기획+디자인+프론트(3플랫폼)+백엔드+인프라+관리자
- MVP이지만 3플랫폼 + 실시간 음성이라 실질 스코프는 대규모
- 디자인 시안 없음 (직접 수행)

**Step 2: 공수**
- 91.8인일 (위 테이블 기반)

**Step 3: 강도(Intensity) 분석**
```
가용 작업일 = 90일 x 5/7 = 64일 (소수점 내림)
강도 = 91.8인일 / 64일 = 1.43
```
- 강도 1.43 -> **불가능** (> 1.3) — 1인으로 물리적 불가, 스코프 축소 필수
- 일당 프리미엄: 정상적이라면 +20~30% 이상이지만, 강도 > 1.3이므로 스코프 축소가 우선

**Step 4: 클라이언트 예산 현실성**
```
클라이언트 기대 일당 = 2,000만 / 90일 = 22.2만원/일
강도 보정 일당 = 22.2만 / 1.43 = 15.5만원/일
```
- 강도 보정 일당 15.5만 -> **타이트** (15~25만 범위 하단)
- 이 예산으로 전체 스코프(91.8인일)를 90일에 수행하면 실질 일당이 15.5만으로, 하한선(15만)에 근접

**Step 5: 제안 금액/기간 결정**

강도 > 1.3 + 타이트 예산 -> 전략: "예산 동일 + MVP만 명시 + 2차 분리"

현실적 MVP 축소:
- WebRTC 자체 구축(30인일) 대신 외부 SaaS(Agora/LiveKit) 활용 시 -> 약 10인일로 축소 (20인일 절감)
- UI/UX 디자인을 최소화(Figma 템플릿 활용) -> 6.5 -> 4인일 (2.5인일 절감)
- MVP 축소 후 총 공수: 약 69.3인일 -> 강도 = 69.3/64 = 1.08 (매우 타이트, 프리미엄 +20~30%)

그러나 이탈도 20% 초과 불가:
- 예산 2,000만 x 1.2 = 2,400만 -> 20% 이탈 한도
- 예산 2,000만 유지, 기간 90일 유지가 이탈도 0%로 안전

**최종 제안:**
- 금액: 2,000만원 / 기간: 90일
- 이유: 강도가 높지만 예산 자체가 타이트하므로 프리미엄을 얹을 수 없음. 대신 지원서에 MVP 범위 한정 + 음성 SaaS 활용을 명시하여 "이 예산에 이 기간으로 가능한 범위"를 제시.
- 제안 일당: 2,000만 / 64가용일 = 31.3만원/일 (캘린더 90일 기준이 아닌 가용 작업일 기준)

**강도 프리미엄:** 적용 대상이나 예산 한도 내에서 흡수 — 이탈도 0%를 유지하면서 MVP 스코프 축소로 실질 강도를 1.08로 낮추는 전략. 강도 1.08은 "매우 타이트"(+20~30%) 구간이나, 프리미엄을 금액에 반영하면 이탈도 20% 초과. 따라서 **프리미엄은 스코프 축소로 대체** — 금액은 동일하되 수행 범위를 줄여 실질 일당을 높이는 효과.

**Step 6: 이탈도 체크**
```
금액 이탈도 = |2,000 - 2,000| / 2,000 = 0% -> 안전
기간 이탈도 = |90 - 90| / 90 = 0% -> 안전
```

**Step 7: 벤치마크 비교**
- 제안 일당: 31.3만원/일 (가용 작업일 기준)
- 신규 파트너 범위 15~33만 -> 이내 (상단)
- 시장 시니어 풀스택 25~40만 -> 이내
- 근거: MVP 축소를 전제로 하면 적정 일당. 전체 스코프 강행 시 실질 일당이 15.5만으로 하한선 수준이므로 비현실적.

**최종 결정 근거:** 예산 2,000만/90일은 전체 스코프(91.8인일) 대비 불가능한 강도(1.43)이나, 음성 SaaS 활용으로 MVP 축소 시 강도 1.08로 수행 가능. 이탈도 0%로 미팅 성사율 극대화하고, 미팅에서 스코프 조율.
